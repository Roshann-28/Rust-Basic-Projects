# Rust Basic Projects

A collection of Rust projects created while learning Rust programming and backend development concepts.

## Projects

### 1. Read_CSV_File

- Reads data from a CSV file using the `csv` crate.
- Demonstrates file handling and CSV parsing in Rust.
- Uses error handling with `anyhow`.

### 2. Compress_File

- Compresses files using Rust.
- Demonstrates file I/O and working with compression libraries.
- Uses error handling with `anyhow`.

### 3. Unzip_file

- Extracts/decompresses zip files using Rust.
- Demonstrates file I/O and working with archive formats.
- Uses error handling with `anyhow`.

More projects will be added as I continue learning Rust.
